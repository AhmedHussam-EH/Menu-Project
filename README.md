# 🍔 Food Palace Menu Project

This is a dynamic menu for "Food Palace" restaurant. It allows users to filter items by category (Breakfast, Lunch, Dinner, etc.) and view details like price, description, and the origin of each dish.

## 🔗 Live Demo
You can check out the live version of this project here:
👉 **[View Live Menu](https://ahmedhussam-eh.github.io/Menu-Project/menu.html)**

## 🛠️ Features
- **Filter by Category:** Easily browse through different types of food and drinks.
- **Responsive Design:** Works on both desktop and mobile devices.
- **Dish Origins:** Learn where each meal comes from (e.g., Pizza from Italy, Pancakes from USA).

## 💻 Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript** (Array mapping, filtering, and DOM manipulation)
